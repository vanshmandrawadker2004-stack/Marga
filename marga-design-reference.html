07 · Rules
✓ Do
  • Use Outfit for all display + body copy
  • Use JetBrains Mono for labels, tags, metadata only
  • Cyan (#00e5ff) for interactive + accent only — never decorative
  • Keep backgrounds layered: base → elevated → overlay
  • Pair large glows only with the map pin or primary CTA
  • Card padding: 24px minimum, 32px standard
✗ Don't
  • Don't use pure #000 or pure #fff backgrounds
  • Don't mix font weights inside a single label element
  • Don't use cyan as a background fill (use dim/subtle variants)
  • Don't stack more than 3 levels of overlay on one screen
  • Don't use border-radius less than 6px on any component
  • Don't use colors outside the defined palette
