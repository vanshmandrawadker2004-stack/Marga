05 — Rules

Usage Guidelines

Cyan — Tech & UI
  • Active states — selected tabs, focused inputs, map pins
  • Primary CTAs — Book, Track, Start ride
  • Live data — real-time indicators, online status
  • App interface — navigation, icons, interactive elements
Warm Amber — Campaign & Moments
  • Event promotions — Instagram posts, banners, posters
  • Editorial headings — campaign titles only, never UI labels
  • Date stamps & sunbursts — special event decorative elements
  • Warm CTAs — Join Event, Register, RSVP
Barlow Condensed — When to Use
  • Instagram posts — vertical type, campaign posters only
  • Event banners — large-format printed or digital assets
  • Never mix with Inter at the same size level
  • Always uppercase, tight leading, weight 700–900 only
Kannada Scripture — .bilingual-label
  • Cultural voice — Noto Sans Kannada renders the sub-label in script
  • Always paired with an English primary label above it
  • Use for route names, event subtitles, brand moments — not UI microcopy
  • Weight 300 (light) only — never bold in the scripture layer
Film Grain — .grain Class
  • Add .grain class to any editorial container for texture
  • Always pair with a dark background or photo overlay
  • Never use on UI cards — editorial/campaign contexts only
  • Opacity is controlled via --grain-opacity (default 0.035)
